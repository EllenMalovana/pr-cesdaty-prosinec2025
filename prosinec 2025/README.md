# Seriály – práce s daty v JavaScriptu

Jednoduchá stránka pro procvičení práce s daty v JavaScriptu:
- Object
- Set
- Map
- filtrace, převod a agregace
- vyhledávání a třídění v českém pořadí

## Spuštění
1. Otevřete `index.html` v prohlížeči
2. Otevřete konzoli (F12)
3. Výsledky jsou vypsány do konzole

## Téma
Datová práce se seriály (např. Fallout, Arcane, Yellowstone).
